# SoundMaster - 视唱练耳训练 App Demo

一个美观、强大且易用的视唱练耳训练应用，采用类多邻国的游戏化设计风格。

## 🎵 功能特性

### 核心功能
- **音符辨识训练**：听音辨音，提升音准辨识能力
- **和弦辨识训练**：听音辨和弦，掌握和弦构成音
- **Web Audio API**：实时音频生成，无需外部音频文件
- **游戏化设计**：连胜系统、积分奖励、等级提升

### 界面设计
- 类多邻国的现代UI设计
- 渐变色彩和流畅动画
- 响应式布局，适配不同屏幕尺寸
- 直观的交互反馈

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本


### 预览生产构建


## 🎯 使用说明

### 音符辨识训练
1. 点击播放按钮听取音符
2. 从12个音符选项中选择正确的答案
3. 系统会立即显示结果和正确答案
4. 保持连胜可获得额外分数奖励

### 和弦辨识训练
1. 点击播放按钮听取和弦
2. 从12个音符选项中选择构成和弦的3个音符
3. 选择完成后点击"检查答案"
4. 系统会显示和弦类型和详细分析

### 游戏化元素
- **连胜**：连续答对获得连胜奖励
- **积分**：正确答案获得积分，和弦训练积分更高
- **等级**：通过经验值积累提升等级
- **进度**：实时显示学习进度和成就

## 🎨 技术栈

- **React 18** - 用户界面框架
- **Vite** - 快速的构建工具
- **Web Audio API** - 音频生成和播放
- **Lucide React** - 图标库
- **CSS3** - 样式和动画

## 🔧 音频引擎特性

- 实时音频合成，无需外部文件
- 支持音符和和弦生成
- 可调音量和播放时长
- 包含多种波形类型（正弦波、方波等）
- 基于标准音高（A4 = 440Hz）
