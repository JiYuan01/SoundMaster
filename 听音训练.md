# 听音训练 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**SoundMaster - 视唱练耳训练**

### 1.2 产品愿景
打造一款美观、强大且易用的移动应用，帮助音乐爱好者通过游戏化的方式系统性地提升视唱练耳能力。

### 1.3 目标用户
- 音乐初学者，希望提高音准辨识能力
- 音乐专业学生，需要日常的练耳训练
- 音乐爱好者，希望提升音乐理论素养
- 准备音乐考试的学生

## 2. 核心功能需求

### 2.1 音符辨识训练
**功能描述：**
- 应用播放单个音符，用户需要辨识并选择正确的音符名称
- 支持不同八度范围的音符
- 提供即时反馈和进度追踪

**具体需求：**
- 音符范围：C3-B5（涵盖常见人声和乐器音域）
- 音符选择：12平均律系统（C, C#, D, D#, E, F, F#, G, G#, A, A#, B）
- 播放选项：纯音、钢琴音色选项
- 难度设置：
  - 初级：自然音阶音符（C, D, E, F, G, A, B）
  - 中级：包含升号
  - 高级：完整半音阶

### 2.2 和弦辨识训练
**功能描述：**
- 应用播放一个和弦，用户需要识别和弦中包含的音符
- 支持不同类型和弦的训练

**具体需求：**
- 和弦类型：
  - 三和弦：大三和弦、小三和弦、增三和弦、减三和弦
  - 七和弦：大七和弦、小七和弦、属七和弦、半减七和弦
- 音符选择：从12个半音中选择和弦包含的音符
- 难度设置：
  - 初级：三和弦辨识
  - 中级：基础七和弦
  - 高级：复杂和弦及转位

## 3. 用户界面设计 (类多邻国风格)

### 3.1 整体设计原则
- **游戏化设计**：采用多邻国的进度条、等级系统、连胜记录等游戏元素
- **简洁直观**：界面简洁，操作简单，降低学习门槛
- **即时反馈**：答题后立即显示结果，正确/错误动画反馈
- **渐进式学习**：难度逐步提升，避免挫败感

### 3.2 主要页面结构

#### 3.2.1 首页
- 用户信息展示（头像、等级、连续学习天数）
- 今日学习进度
- 快速开始按钮
- 学习路径展示

#### 3.2.2 训练页面
- 音频播放区域（大图标播放按钮）
- 答题选项区域
- 提示按钮（减少金币）
- 进度指示器
- 音量调节

#### 3.2.3 结果页面
- 得分显示
- 正确率统计
- 错误分析
- 继续学习/重试按钮
- 社交分享功能

#### 3.2.4 学习路径页面
- 单元解锁进度
- 成就徽章展示
- 学习统计图表

### 3.3 色彩设计
- 主色调：绿色系（成功、进步）
- 辅助色：橙色（警告、错误）、蓝色（信息）
- 背景色：白色和浅灰色交替
- 强调色：用于重要提示和按钮

## 4. 交互设计

### 4.1 音符辨识流程
1. 进入训练页面
2. 点击播放按钮听取音符
3. 可重复播放（限制次数或消耗金币）
4. 从12个音符选项中选择答案
5. 提交答案
6. 查看结果和解析
7. 进入下一题

### 4.2 和弦辨识流程
1. 进入和弦训练页面
2. 播放和弦音
3. 分析和弦音高关系
4. 从多选选项中选择组成音符
5. 提交答案
6. 查看和弦分析和错误纠正
7. 继续训练

## 5. 技术需求

### 5.1 音频生成
- 支持实时音频合成（不依赖外部音频文件）
- 音频质量：44.1kHz/16bit 或更高
- 支持多种乐器音色
- 低延迟音频播放

### 5.2 性能要求
- 应用启动时间 < 3秒
- 音频播放延迟 < 100ms
- 电池消耗优化（音频播放和界面渲染）
- 内存占用 < 150MB

## 6. 用户数据管理

### 6.1 学习进度
- 完成题目数量统计
- 正确率计算
- 学习时长记录
- 弱点音符/和弦分析

### 6.2 成就系统
- 连续学习天数奖励
- 等级提升系统
- 成就徽章收集
- 排行榜功能（可选）

### 6.3 数据同步
- 本地数据存储
- 云端备份（可选）
- 跨设备数据同步（需要登录）
